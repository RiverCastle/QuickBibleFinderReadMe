## 설명

이 프로젝트는 **교회 방송팀에서 실시간으로 성경 구절을 찾아 시청자들에게 표시**하기 위해 개발되었습니다.  
주요 기능으로는 **사용자로 하여금 실시간으로 성경 구절을 찾아 시정자들에게 표시하는 기능**을 제공합니다. 이 프로젝트는 **사전에 협의되지 않은 성경 구절이 시청자들에게 표시될 수 없는 문제**를 해결하기 위해 설계되었습니다.
